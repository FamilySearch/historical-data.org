What is historical-data.org?

This site provides a collection of schemas, i.e., html tags, that webmasters can use to markup their historical and genealogical information in a consistent way.

Many sites are generated from structured data, which is often stored in databases. When this data is formatted into HTML, it becomes very difficult to recover the original structured data. Many applications, especially search engines, can benefit greatly from direct access to this structured data. On-page markup enables search engines to understand the information on web pages and provide richer search results in order to make it easier for users to find relevant information on the web. Markup can also enable new tools and applications that make use of the structure.

Date Note

Schema.org, Microformats, and Microdata all rely on the ISO 8601 date format, which cannot support approximate dates. Review the Library of Congress's Extended Date and Time Format, which extends ISO 8601 and allows uncertain and approximate dates.
