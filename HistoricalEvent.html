Thing > Event > Historical Event

Event type: Historical event.
Property Expected Type Description
Properties from Thing
description Text A short description of the item.
image URL URL of an image of the item.
name Text The name of the item.
url Text URL of the item.
Properties from Event
duration Duration The duration of the item (movie, audio recording, event, etc.) in ISO 8601 date format (not required for single-day events).
endDate Date The end date and time of the event in ISO 8601 date format (not required for single-day events).
location Place or PostalAddress The location of the event or organization.
startDate Date The start date and time of the event (in ISO 8601 date format).
Properties from Historical Event
agent Person Person who is responsible for creating/maintaining information about the event.
attendees Historical Person A historical person who was present the event. (Use Historical Person instead of Person)
documents Historical Document A historical document about or created at the event.
media Media Object Image, video or audio object associated with this event.
rev Date Date of the most recent revision to event metadata.
subEvents Historical Event Events that are a part of this event. For example, a conference event includes many presentations, each are subEvents of the conference. (Use Historical Event instead of Event)
superEvent Historical Event An event that this event is a part of. For example, a collection of individual music performances might each have a music festival as their superEvent. (Use Historical Event instead of Event)
type Type Type of event (e.g. reunion, employment, residence, education, etc.)
